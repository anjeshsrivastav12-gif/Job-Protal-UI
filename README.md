# Job Portal Web Application
## Project Overview
This is an Advanced Job Portal Web Application built using HTML, CSS, and JavaScript.
It allows users to search jobs, save jobs, apply for jobs, and manage their profile.
Recruiters can post jobs through a dashboard.
## Features- Job search functionality- Filter jobs by location and salary- Save jobs- Apply for jobs via form- Login & Signup system using LocalStorage- User profile with stats- Recruiter dashboard for posting jobs- Job details popup (modal)- Dark mode toggle- Responsive UI
## Technologies Used- HTML- CSS- JavaScript- LocalStorage
## Project Structure- index.html- style.css- script.js
## How to Run
1. Open index.html in browser
2. No installation required
